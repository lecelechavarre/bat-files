# Batch Files Repository

This repository contains various batch scripts to automate tasks and enhance productivity. Each script is designed to perform specific functions that can simplify operations within the Windows environment.

## Included Scripts:

1. **backup.bat** - A script for creating backups of important files.
2. **clean_temp.bat** - A script to clean temporary files from the system.
3. **auto_shutdown.bat** - A script to schedule shutdowns.

You can run any of these scripts from the command prompt or by double-clicking them in Windows Explorer. Please review the code and modify it according to your needs before use.
